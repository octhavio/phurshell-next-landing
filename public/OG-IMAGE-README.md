# Open Graph Image

## Arquivo Necessário
Criar arquivo: `og-image.png`

## Especificações
- **Dimensões**: 1200x630 pixels (proporção 1.91:1)
- **Formato**: PNG ou JPG
- **Tamanho máximo**: 8 MB (recomendado: menos de 300 KB)
- **Posição**: `/public/og-image.png`

## Conteúdo Sugerido
- Logo da Phurshell
- Texto: "Desenvolvimento de Apps Sob Medida"
- Subtexto: "Transformamos ideias em produtos digitais de alta performance"
- Cores da marca (laranja #FF5100 e dark #19191A)
- Background limpo e profissional

## Ferramentas Recomendadas
- Figma: https://figma.com
- Canva: https://canva.com
- OG Image Generator: https://og-playground.vercel.app/

## Teste
Após criar a imagem, teste em:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
